constraints are the following:

                 vx = 0,         vz = 0,          Ï‰x = 0,             Ï‰z = 0,              fy = 0,        Âµy = 0.

are chosen then as:

           fx = 0,       fz = 0,            Âµx = 0,         Âµz = 0,                vy = vd,y (t),         Ï‰y = Ï‰d,y (t).
